This project is a smart home automation system that enables control of lights, fans, and appliances via a custom Android app over WiFi. It uses an ESP8266 microcontroller, sensors for feedback, and relays for device control.

  • Real-time control using mobile app
  • Built with Arduino IDE, Firebase, and ESP8266
  • Expandable to voice assistants and security systems
